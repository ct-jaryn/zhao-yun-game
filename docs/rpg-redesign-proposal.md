# 三国赵云传 · 长坂坡 — 长期 RPG 化改造方案

> 目标：将当前「单局幸存者-like」改造成可长期游玩、具备角色管理、多英雄养成、局外成长的轻度 RPG。

---

## 1. 设计目标

| 维度 | 现状 | 目标 |
|---|---|---|
| 单局时长 | 10~15 分钟通关 4 章 | 保留快速通关，同时提供 30~60 分钟无尽/挑战 |
| 成长 | 单局内升级，出局重置 | 局外账号等级 + 英雄养成 + 装备收藏 |
| 重复游玩 | 4 章 + 2 皮肤 | 多英雄、多难度、随机 Build、每日挑战 |
| 角色管理 | 无 | 新增「角色管理面板」作为游戏大厅核心 |
| 付费/留存点 | 第四章广告锁屏 | 皮肤、英雄、战令式「军功簿」、抽卡式寻访 |

---

## 2. 核心循环（Core Loops）

### 2.1 日常循环（5~15 分钟）

```
登录 → 领取每日/军功簿奖励 → 选择模式 → 单局战斗 →
结算（经验/装备/将魂/战功） → 强化英雄/装备 → 角色管理面板
```

### 2.2 周常循环（数小时）

```
完成每日挑战 → 积累「将魂」解锁新英雄/天赋 →
冲击无尽排行榜 → 刷指定装备/套装 → 挑战更高难度
```

### 2.3 长期循环（数周）

```
账号升阶（校尉→将军→大都督） → 英雄满级 →
收集全装备图鉴 → 全英雄通关修罗难度 → 赛季重置/轮回
```

---

## 3. 账号体系：将魂与军阶

### 3.1 账号经验：军功

- 每局根据「击杀数、得分、通关章节、难度系数」结算军功。
- 军功提升账号「军阶」（1~60 级）。
- 军阶解锁：英雄槽位、天赋页、装备仓库格、每日挑战次数。

### 3.2 通用货币

| 货币 | 获取 | 用途 |
|---|---|---|
| **铜币** | 击杀小兵、日常任务 | 强化装备、刷新商店 |
| **将魂** | Boss 掉落、成就、无尽层数 | 解锁英雄、升星、购买皮肤 |
| **战功** | 通关章节、挑战模式 | 兑换高级装备箱、限时增益 |
| **元宝** | 充值/广告/活动（可选） | 抽卡、皮肤、快速复活 |

### 3.3 军功簿（Battle Pass）

- 免费线与付费线并行。
- 奖励：将魂、铜币、皮肤碎片、限定头像框。
- 赛季主题（如「虎牢关赛季」）。

---

## 4. 英雄系统（Hero Roster）

### 4.1 初始英雄

| 英雄 | 定位 | 特色 | 解锁条件 |
|---|---|---|---|
| **赵云** | 均衡战士 | 龙胆枪、连击增伤 | 初始 |
| **貂蝉** | 辅助/控制 | 魅惑、治疗、范围减速 | 通关第一章 |
| **典韦** | 狂战士 | 高伤低防、嗜血 | 将魂 ×500 |
| **吕布** | 爆发输出 | 方天画戟、真实伤害 | 将魂 ×1500 或击败指定 Boss |
| **许褚** | 坦克 | 嘲讽、护盾、反伤 | 将魂 ×1200 |
| **马超** | 机动刺客 | 高速突刺、暴击 | 赛季/活动 |

### 4.2 英雄属性（与现有 `Player` 对应）

```ts
interface Hero {
  id: string;              // zhaoyun, diaochan, dianwei...
  name: string;
  rarity: number;          // 1~5 星
  level: number;           // 1~60
  exp: number;
  stars: number;           // 升星，提升基础成长
  skin: string;            // classic / mecha / 赛季皮肤
  affinity: string;        // 蜀/魏/群（影响阵营羁绊）

  baseStats: {
    hp: number;
    mp: number;
    atk: number;
    def: number;
    crit: number;
    spd: number;
  };

  skillBuild: number[];    // 5 个技能槽的「强化分支」索引
  talentTree: TalentNode[];
  equipment: Record<EquipSlot, EquipItem>;
}
```

### 4.3 英雄升星与升级

- **升级**：消耗铜币 + 经验丹（或重复通关），提升基础属性。
- **升星**：消耗英雄碎片（将魂商店/抽卡），解锁新天赋、新皮肤配色、被动技能。
- **天赋树**：每个英雄 3 条分支（如赵云：龙胆/枪神/无双），消耗将魂点亮。

## 5. 角色管理面板（Character Management UI）

角色管理面板取代现有「开始菜单」成为游戏主大厅。面板不进入 Phaser，使用 HTML/CSS 实现。

### 5.1 面板布局

```
┌─────────────────────────────────────────────────────────────┐
│  [头像] 军阶 Lv.12  铜币 8,520  将魂 340  战功 120    [设置]  │
├──────────┬──────────────────────────────────────────────────┤
│          │                                                  │
│  英雄    │   英雄立绘 / 皮肤展示 / 模型预览                 │
│  列表    │                                                  │
│          ├──────────────────────────────────────────────────┤
│  [赵云]  │   属性  |  装备  |  技能  |  天赋  |  成就       │
│  [貂蝉]  │                                                  │
│  [典韦]  │   当前选中 Tab 内容                              │
│  [+]     │                                                  │
│          │                                                  │
├──────────┴──────────────────────────────────────────────────┤
│  [剧情模式]  [无尽模式]  [每日挑战]  [多人/异步]  [商店]      │
└─────────────────────────────────────────────────────────────┘
```

### 5.2 左侧英雄列表

- 已解锁英雄显示头像、等级、战力。
- 未解锁英雄显示剪影、解锁条件（将魂/通关）。
- 点击切换当前操作英雄。
- 长按/右键查看英雄详情。

### 5.3 右侧 Tab 页

#### Tab 1：属性

显示当前英雄所有属性与战斗力：

```
战力  12,450
生命  1,850   攻击  285   防御  120
法力  320     暴击  18%   速度  235

被动：龙胆 — 生命值低于 30% 时，伤害 +25%
```

#### Tab 2：装备

扩展现有 `equipPanel`：

- 5 个装备槽（武器/铠甲/头盔/靴子/饰品）。
- 点击槽位弹出「装备选择/强化/分解」。
- 显示套装效果：「龙胆套装 3/5：大招伤害 +20%」。
- 装备战力对比：新装备 vs 当前装备。

#### Tab 3：技能

- 5 个技能图标 + 当前强化分支。
- 点击技能进入「技能强化」：消耗铜币/战功提升等级或选择分支。
- 可保存 2~3 套技能 Build，局内根据模式切换。

#### Tab 4：天赋

- 树状天赋图，分 3 支。
- 每个节点：消耗将魂点亮，提供属性或机制改变。
- 支持一键重置（消耗少量元宝/铜币）。

#### Tab 5：成就/战绩

- 该英雄通关记录：最高章节、最高无尽层、最高连击、总击杀。
- 成就徽章：「单骑救主」「万人敌」「常胜将军」。

### 5.4 底部模式入口

| 按钮 | 说明 |
|---|---|
| **剧情模式** | 现有 4 章 + 新增后续章节，分普通/困难/修罗 |
| **无尽模式** | 生存挑战，难度随时间提升，排行榜 |
| **每日挑战** | 每日随机词条（如「只能用普攻」「敌人双倍血」） |
| **将魂试炼** | 限时刷指定资源 |
| **商店** | 将魂/战功兑换、装备箱 |

## 6. 装备与收藏系统

### 6.1 装备改造

| 功能 | 说明 | 消耗 |
|---|---|---|
| **强化** | 提升装备基础数值 | 铜币 + 强化石 |
| **精炼** | 提升品质星级（绿→蓝→紫→金） | 同名装备 / 精铁 |
| **镶嵌** | 宝石/符文槽，附加属性 | 宝石 |
| **洗练** | 重 roll 词条 | 洗练石 |
| **分解** | 回收为材料 | — |

### 6.2 装备图鉴

- 所有获得过的高品质装备进入图鉴。
- 集齐套装激活账号级属性加成。
- 图鉴进度给将魂/头像框奖励。

### 6.3 装备背包

- 仓库容量随军阶增加。
- 支持一键筛选、排序、锁定。
- 超出容量可花元宝扩展或用仓库箱。

## 7. 单局战斗改造

### 7.1 出战前配置

```
选择英雄 → 选择皮肤 → 选择技能 Build → 选择难度 → 选择增益道具 → 开始
```

### 7.2 难度系数

```js
const DIFFICULTY = {
  normal:  { enemyHp: 1.0, enemyAtk: 1.0, exp: 1.0, reward: 1.0 },
  hard:    { enemyHp: 1.6, enemyAtk: 1.4, exp: 1.2, reward: 1.3 },
  hell:    { enemyHp: 2.5, enemyAtk: 2.0, exp: 1.5, reward: 1.8 },
  reincarnation: { enemyHp: 4.0, enemyAtk: 3.0, exp: 2.0, reward: 2.5, affixes: true }
};
```

### 7.3 局内成长

保留现有升级三选一，但奖励池扩展为：

1. 属性强化（现有）
2. 技能强化（改变技能形态）
3. 临时神器（仅本局生效，如「青釭剑：对精英 +30% 伤害」）
4. 恢复/增益

### 7.4 章节扩展

在原 4 章基础上新增：

| 章节 | 主题 | Boss | 新机制 |
|---|---|---|---|
| 5 | 赤壁疑云 | 张辽 | 火海地形、风向影响 |
| 6 | 华容道 | 曹操（真身） | 限时追击/放走选择 |
| 7 | 定军山 | 夏侯渊 | 高地视野、援军 |
| 8 | 五丈原 | 司马懿 | 阵法变化 |

## 8. 无尽模式设计

### 8.1 基础规则

- 单张大地图，敌人无限刷新。
- 每 60 秒为一「波次」，波次结束选择一次奖励。
- 难度随波次指数增长。
- 死亡即结束，按存活时间和击杀数排名。

### 8.2 波次事件

| 波次 | 事件 |
|---|---|
| 5 的倍数 | 精英怪潮 |
| 10 的倍数 | Boss 降临 |
| 20 的倍数 | 双 Boss |
| 随机 | 特殊事件（护送/防守/限时击杀） |

### 8.3 无尽排行榜

- 本地榜单 + 可选服务端榜单。
- 分英雄、分难度榜单。
- 每周重置，前 N 名获得称号/头像框。

## 9. 每日挑战与活动

### 9.1 每日挑战

- 每天 3 个随机任务：
  - 「只用普攻通关第一章」
  - 「30 秒内击杀吕布」
  - 「单局连击达到 100」
- 奖励：将魂、铜币、装备箱。

### 9.2 限时活动

- 「虎牢关之战」：限定时间内击杀尽可能多的敌人。
- 「宛城突围」：保护 NPC 到达撤离点。
- 「吕布讨伐令」：全服累计伤害解锁奖励。

---

## 10. 赛季与轮回

### 10.1 赛季制

- 每 6~8 周一个赛季。
- 赛季主题：新英雄、新皮肤、新章节、新天赋。
- 赛季结束后，部分进度保留，部分重置。

### 10.2 轮回系统（防止数值膨胀）

当账号达到满级或英雄达到上限后，可「轮回」：
- 重置军阶等级至 1，但保留英雄星级、皮肤、图鉴。
- 获得永久「天命点数」，用于账号级被动树。
- 轮回次数显示为称号（一转转世、二转涅槃）。

## 12. 分阶段落地计划

### 第一阶段：角色管理面板 + 账号数据（2~3 周）

- [ ] 新建 `save/` 数据层，用 localStorage 持久化。
- [ ] 实现角色管理面板 HTML/CSS，替代开始菜单。
- [ ] 英雄列表、属性 Tab、装备 Tab。
- [ ] 战斗前读取英雄属性进入 `GameController`。
- [ ] 战斗结算回写经验、铜币、掉落。

### 第二阶段：多英雄 + 天赋（2~3 周）

- [ ] 解锁貂蝉、典韦两个新英雄。
- [ ] 每个英雄实现差异化基础属性与被动。
- [ ] 天赋树 UI 与数据。

### 第三阶段：装备深度 + 无尽模式（3~4 周）

- [ ] 装备强化、精炼、镶嵌。
- [ ] 套装效果与图鉴。
- [ ] 无尽模式改造。
- [ ] 本地排行榜。

### 第四阶段：运营内容（持续）

- [ ] 每日挑战、军功簿、赛季。
- [ ] 新英雄/新皮肤/新章节。
- [ ] 云存档与排行榜服务端。

## 13. 关键设计原则

1. **渐进式改造**：先不改变核心战斗，只加局外面板与数据持久化。
2. **英雄差异足够大**：避免变成换皮，每个英雄改变操作节奏。
3. **付费不破坏平衡**：付费点集中在皮肤、英雄解锁加速、仓库扩展、Battle Pass。
4. **单局仍有爽感**：幸存者-like 的核心割草感不能丢，RPG 是放大器。
5. **离线优先**：先保证本地存档完整，云存档作为可选增强。

| 现有模块 | RPG 化改造 |

| `api.js` | 未来接入云存档、排行榜、支付 |

## 15. 结论

本方案以「角色管理面板」为入口，围绕**多英雄养成 + 账号成长 + 装备收藏 + 无尽/挑战模式**构建长期玩法。第一阶段即可让玩家看到自己的账号在成长，后续逐步加深 Build 维度。整个改造可以分阶段平滑落地，不会一次性推翻现有战斗体验。
